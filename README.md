# 🏪 快消与零售执行：售点评估方法论资产库

本仓库用于整理快消（FMCG）与零售执行领域中，关于**售点评估（Store Scoring）、潜在价值预测与选址模型**的核心知识与技术。

---

## 📌 知识框架目录

### 1. 售点评估模型分类
- **基于规则的打分卡（Scorecard）**：潜力（Potential）、执行（Execution）、表现（Performance）三维打分。
- **层次分析法（AHP）**：将专家主观经验量化为逻辑自洽的权重矩阵。
- **机器学习模型**：利用 Random Forest 和 XGBoost 进行数据驱动的评分与销量预测。

### 2. 核心算法原理
- **Random Forest（随机森林）**：多决策树随机抽样 + 投票机制，容错率高，可输出特征重要性（Feature Importance）。
- **XGBoost（极限梯度提升）**：串行纠错机制，预测精度极高，适合复杂的售点销量预测。

### 3. 空白门店（无历史数据）评估方案
- **Look-alike 相似匹配**：基于 K-NN 算法寻找“双胞胎”已合作门店。
- **重力模型（Gravity Model, 1931）**：基于规模成正比、距离平方成反比的传统选址引力模型。
- **哈夫模型（Huff Model, 1963）**：引入概率论与竞争吸引力，计算空白门店的抓客概率 $P_{ij}$。

### 4. 现代前沿评估技术
- **时空图神经网络（Spatial GNN / ST-GNN）**：捕捉动态人流与复杂道路网。
- **计算机视觉（CV）与卫星/街景图**：自动识别门头宽度、排面及区域经济指数。
- **手机信令（Cellular Signaling Data）**：通过基站脱敏数据，绘制真实客流热力图、分析客流来源（OD 分析）与常住/过客画像。

### 5. 总结：新旧模型的演进对比

| 评估维度 | 传统模型（重力/哈夫模型） | 现代模型（GNN / Transformer / 空间 AI） |
| :--- | :--- | :--- |
| **数据源** | 静态数据（营业面积、直线距离、人口普查） | 动态大数据（手机信令、热力图、街景图像、POI） |
| **评估维度** | 假设人流是均匀且理性的 | 捕捉非理性人流、时段变化、天气与复杂交通 |
| **计算方式** | 简单的物理/概率公式 | 机器学习、深度神经网络、时空图计算 |
| **应用阶段** | 适用于早期**宏观商圈规划** | 适用于现代**微观售点评估、极精细化资源投放** |
